- **Squeezed-Vacuum MZI with Parity Measurement** (#20260625-ext) — Full implementation of parity measurement for squeezed-vacuum MZI (`reports/20260625/squeezed_vacuum_parity.py`, 900 LOC). 52 tests passing.
  - **Physical finding**: Parity CFI is **zero** with `skip_bs1=True` (contrary to the original hypothesis). The SV state Σ c_n |2n,0⟩ has total-photon-number superselection: after BS2, each output Fock state |n₁,n₂⟩ couples to exactly one input component n=(n₁+n₂)/2. Since Π=(-1)^{n₂} is diagonal in the Fock basis, no coherences between different n produce ω-dependent interference — ⟨Π⟩ = 1/cosh(r) is constant, giving F_C^Π = 0 just like number-difference.
  - With `skip_bs1=False` (BS1 applied before phase shift), parity CFI **saturates** the QFI bound: Δω_C/Δω_Q ≈ 1.001 across all ⟨N⟩=1–20. BS1 creates path entanglement that allows the J_z phase shift to produce ω-dependent parity oscillations.
  - **Results generated**: Full sweep over ⟨N⟩=1–20, ω=0.1–5.0 (step 0.1, 50 pts). Parquet file: `raw_data/20260625-sv_parity_sensitivity.parquet` (20 R × 50 ω = 1000 rows, 65KB). Figures: `figures/20260625-sv_parity_delta_omega_comparison.svg` (62KB), `figures/20260625-sv_parity_scaling.svg` (54KB).
  - **Scaling finding**: Fitted exponent α = −0.492 (SQL-like), **not** Heisenberg (−1.0) as the report hypothesised. With `skip_bs1=False`, BS1 transforms the SV state into a probe whose J_z variance scales as ⟨N⟩ (SQL) rather than ⟨N⟩² (Heisenberg). The analytical formula F_Q = 2 H_t² ⟨N⟩(⟨N⟩+1) applies only to the pre-BS1 state — the numerical QFI (computed from the state at the point of phase accumulation, after BS1) gives SQL scaling. Parity measurement is **optimal** (saturates this QFI bound), but the achievable bound is SQL.
  - Number-difference measurement (skip_bs1=False): F_C ≈ 10⁻²² at ⟨N⟩=2, confirming parity recovers sensitivity where number-difference gave zero.
  - Cramér-Rao bound satisfied at all 1000 operating points (0 violations).
  - Updated `generate_single_omega_scan` to default `skip_bs1=False` (the only configuration where parity measurement recovers sensitivity).
  - **Report compiled**: Hypotheses revised — Hypothesis 2 (Heisenberg scaling) marked FAIL with α=−0.4923 (SQL). Theoretical model corrected to distinguish pre-BS1 input QFI from post-BS1 probe QFI. Results section updated with actual data (1000-row Parquet, 2 SVG figures). Success Criteria: 5 PASS, 1 PARTIAL (truncation), 1 FAIL (Heisenberg scaling). Conclusions rewritten to explain SQL scaling origin. Main report (#20260625) updated with parity findings. Findings document updated (entry 31). All 191 tests pass. See #20260625.
